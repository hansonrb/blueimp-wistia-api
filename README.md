[https://blueimp-wistia-api.herokuapp.com/](https://blueimp-wistia-api.herokuapp.com/)

## Description

This is a AngularJS exam where the idea is communicate the *wistia-api* by sending them videos.

## Setup

If you want install...

```
$ npm install
```

You can run...

### Local

```
$ npm start
```

### Heroku (simulation)

```
$ npm run heroku-start
```
